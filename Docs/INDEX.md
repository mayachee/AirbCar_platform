## 🛠️ TOOLS SETUP

### **Postman Setup**
1. Download Postman from [getpostman.com](https://getpostman.com)
2. Import: [POSTMAN_COLLECTION_COMPLETE.json](POSTMAN_COLLECTION_COMPLETE.json)
3. Set environment `base_url` = `http://localhost:8000`
4. Run tests: Collection → Run

See: [TESTING_SUMMARY.md - Tools](TESTING_SUMMARY.md#tools-youll-need)

### **Backend Setup**
```bash
docker compose up -d web
docker compose exec web python manage.py migrate
curl http://localhost:8000/api/health/
```

See: [API_TESTING_GUIDE.md - Quick Start](API_TESTING_GUIDE.md#quick-start)
